READ before execution.

-> To run the Season Predictor, navigate to the Football-Season-Prediction directory.
-> You may need to install the streamlit module beforehand: use "pip install streamlit"
-> Once installed you can run "streamlit run app.py" in your terminal. This will start the data loading process. -> You will be redirected to your browser, if connection is formed, you will be able to select the League and Number of simulations.
-> The simulation will be run, and it will produce the predicted table based on historical data (present in /data directory)
